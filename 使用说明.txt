= 第一步：下载并安装Tampermonkey =

■ Chrome浏览器用户：
1. 打开Chrome浏览器
2. 复制这个网址到地址栏：https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

4. 点击蓝色的"添加至Chrome"按钮
5. 弹出确认窗口，点击"添加扩展程序"

= 第二步：添加自动登录脚本 =

1. 点击浏览器右上角的Tampermonkey黑色小方块图标
2. 在弹出的菜单中，点击"管理面板"
3. 会打开一个新的页面，点击左上角的"+"号（新建脚本按钮）
4. 这时会出现一个编辑器，里面有一些默认代码
5. 按Ctrl+A全选所有内容，然后按Delete键删除
6. 找到tampermonkey_autofill.js文件，用鼠标直接拖到编辑器的空白区域
7. 文件内容会自动填入编辑器
8. 按Ctrl+S保存脚本
9. 左上角会显示一个绿色的圆点，表示脚本已保存并启用

= 第三步：修改你的登录信息 =

1. 在刚才的编辑器中，找到代码开头的配置区域（大约在第15行左右）
2. 找到这几行：
   username: '你的学工号',
   password: '你的密码',
   isp: '中国移动',

3. 修改为你的实际信息：
   username: '231080906218',     ← 改为你的真实学工号
   password: '你的真实密码',      ← 改为你的真实密码
   isp: '中国联通',              ← 改为你的运营商

4. 运营商选项只能是以下四种之一：
   中国移动
   中国联通
   中国电信

5. 修改完成后，再次按Ctrl+S保存

= 使用方法 =

1. 确保电脑已连接到校园网JCI
##通常来说只要你上一次连接WiFi时勾选了自动连接，就不需要你进行下面的步骤了，系统会自动弹出校园网浏览器登录页面。若没有进行下面的步骤。
2. 打开浏览器，在地址栏输入：http://172.29.0.2/a79.htm  
3. 按回车键访问登录页面
4. 等待2-3秒，脚本会自动填写你的学号、密码和运营商
5. 自动点击登录按钮
6. 登录成功后，页面会自动关闭
7. 现在可以正常上网了

= 注意事项 =

- 第一次设置完成后，以后每次只需要访问登录页面即可自动登录
- 如果脚本不工作，检查Tampermonkey图标是否已启用；点击管理扩展程序，点击Tampermonkey(篡改猴)的详情，勾选‘允许运行用户脚本’。

- 修改配置后记得保存（Ctrl+S）
- 学号和密码要用英文单引号包围，不要随便点击：‘Enter’，若代码出问题，全部删除，重新复制。
